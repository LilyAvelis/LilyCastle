Твоя задача: Создать MVP лендинга для ниши "Фудфлористика" (съедобные букеты).

СТУКТУРА ПРОЕКТА И СТЕК:

1. Фреймворк: Nuxt 3 (Vue 3 Composition API, <script setup lang="ts">).
2. Стилизация: Tailwind CSS.
3. Иконки: Используй библиотеку `lucide-vue-next`.
4. Шрифт: Предлагаю использовать системные шрифты sans-serif, настроенные через Tailwind.

ТРЕБОВАНИЯ К ДИЗАЙНУ (ИЗ ТЗ ЗАКАЗЧИКА):

- Эстетика: Минимализм, "много воздуха".
- Цветовая палитра: Пастельные тона.
  - Фон: bg-stone-50 (слоновая кость/светлый камень).
  - Текст: text-stone-800.
  - Акценты: text-rose-400 или bg-rose-400 (нежный розовый).
- Типографика: Четкая иерархия, крупные заголовки, легкий основной текст.

ЗАДАЧА:
Напиши код для следующих компонентов (в отдельных блоках) и собери их на странице `pages/index.vue`.

НЕОБХОДИМЫЕ КОМПОНЕНТЫ:

1. components/AppHeader.vue

   - Логотип (текст "SweetBouquet").
   - Навигация (Каталог, О нас, Контакты).
   - Кнопка "Заказать" (акцентная).
   - Сделай "липким" (sticky) с легкой тенью/блюром.

2. components/HeroSection.vue

   - Левая часть: Заголовок "Вкусные эмоции в каждом букете", подзаголовок, кнопка CTA "Выбрать букет".
   - Правая часть: Плейсхолдер под изображение (используй серый div с иконкой или placeholder url), скругленные углы (rounded-2xl).

3. components/FeatureList.vue

   - Секция с 3-мя преимуществами (Свежесть, Авторский дизайн, Доставка).
   - Используй иконки из Lucide (Leaf, Star, Truck).
   - Оформление: Grid-сетка, минималистичные карточки без рамок.

4. components/ProductCatalog.vue

   - Заголовок секции "Популярные композиции".
   - Сетка (Grid) карточек товаров.
   - Карточка товара должна быть отдельным микро-компонентом (внутри этого файла или отдельно), принимающим props: title, price, image, category.
   - Данные: Создай массив mock-объектов (5-6 товаров) прямо в скрипте.

5. components/AppFooter.vue
   - Простой подвал с контактами и копирайтом.

ФИНАЛЬНАЯ СБОРКА:

- Покажи код файла `pages/index.vue`, где ты импортируешь и расставляешь эти компоненты в правильном порядке.

ВАЖНО:

- Пиши чистый код.
- Используй `defineProps` для передаваемых данных.
- Делай адаптив (mobile-first классы Tailwind: md:flex, lg:grid-cols-3 и т.д.).
- Не используй сложные внешние зависимости, только нативный Vue/Nuxt функционал.
