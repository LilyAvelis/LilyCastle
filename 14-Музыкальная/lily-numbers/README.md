# Lily Numbers Research

Исследование рядов Лилии - многократных кумулятивных сумм простых чисел (с включением 1).

## Задача

Изучить поведение рядов, полученных путем n-кратного кумулятивного суммирования последовательности простых чисел, где 1 считается первым "простым" числом (Аксиома Лилии).

## Файлы

- `data_primes.py` - генерация простых чисел
- `data_series.py` - вычисление рядов Лилии L_n
- `analysis_core.py` - анализ роста и поиск константы ускорения
- `visualization_core.py` - визуализация результатов (требует matplotlib)
- `visualization_dashboard.py` - красивый дашборд с графиком константы ускорения
- `ui_desktop.py` - интерактивный UI для исследования (tkinter + matplotlib)
- `ui_web.py` - веб-приложение Streamlit для интерактивной визуализации
- `analysis_asymptotic.py` - анализ асимптотического поведения A(k) при k → ∞
- `analysis_formula.py` - детальное исследование асимптотической формулы

## Запуск

```bash
# Красивый график константы ускорения
python visualization_dashboard.py

# Интерактивный UI (tkinter)
python ui_desktop.py

# Веб-приложение (Streamlit)
streamlit run ui_web.py

# Исследование асимптотической формулы
python analysis_formula.py

# Асимптотический анализ
python analysis_asymptotic.py
```

## Результаты исследования

- ✅ **Константа ускорения существует** и зависит от количества простых чисел!
- ✅ **Стремится к 1 при k→∞** - асимптотическое поведение подтверждено
- ✅ **Асимптотическая формула**: A(k) ≈ 1 - 0.003526 / k^(-0.1301)
- ✅ **Рациональные приближения** с высокой точностью
- ✅ **Экспоненциальная сходимость** с показателем ≈ 0.0024
- ✅ **Стабилизация** начинается с n ≈ 30-40

### Асимптотическое поведение A(k) при k→∞

**Экспериментально подтверждено**: A(k) → 1 при увеличении количества простых чисел k.

| k (простых) | A(k)     | Отклонение от 1 | Стабильность |
| ----------- | -------- | --------------- | ------------ |
| 50          | 0.989997 | 0.010003        | 4.26e-04     |
| 100         | 0.992569 | 0.007431        | 1.97e-04     |
| 200         | 0.993403 | 0.006597        | 1.22e-04     |
| 500         | 0.993309 | 0.006691        | 8.96e-05     |
| 1000        | 0.993693 | 0.006307        | 6.58e-05     |

**Теоретическое объяснение**: При k→∞ базовый ряд простых чисел ведет себя как непрерывная функция, для которой многократное интегрирование дает константу ускорения = 1.

### Зависимость от количества простых (детальный анализ)

| Простых | Предел   | Рациональное приближение |
| ------- | -------- | ------------------------ |
| 30      | 0.996562 | 12457/12500              |
| 50      | 0.995044 | 6219/6250                |
| 80      | 0.993403 | 4967/5000                |
| 100     | 0.992569 | 99257/100000             |
| 150     | 0.991024 | 49551/50000              |
| 200     | 0.994876 | 101537/102060            |

**Вывод:** Константа ускорения A(k) является функцией от количества простых чисел k в базовом ряде и асимптотически приближается к 1.

## Гипотезы

1. **Функция от количества простых**: A(k) → L(k), где k - количество простых
2. **Предел при k→∞**: Возможно существует фундаментальная константа ∈ (0.99, 1)
3. **Рациональность**: Значения A(k) могут быть рациональными для конечных k
4. **Теоретико-числовой смысл**: Связь с распределением простых чисел
5. **Аналитическое выражение**: Существует формула для A(k)
6. **Обобщение**: Аналогичные константы для других последовательностей
